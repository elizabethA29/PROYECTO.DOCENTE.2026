ACTIVIDADES

Actividad: Desarrollo para el movimiento de la araña.

En este apartado se presenta  el código.

## 1. Inclusión de Librerías y Definiciones

```cpp

#include <setjmp.h>

#include "GoBLE.h"

#include "MiniKame.h"

```

- Se incluyen las librerías necesarias: `setjmp.h` para manejo de saltos, `GoBLE.h` para la comunicación Bluetooth y `MiniKame.h` para controlar el robot.

 

### Definiciones de Macros

 

```cpp

#define __DEBUG__

//#define __HC_SR04__

```

- `__DEBUG__`: Activa el modo de depuración.

- `__HC_SR04__`: Permite el uso del sensor ultrasónico HC-SR04 (actualmente comentado).

 

### Definiciones de Pines y Comandos

 

```cpp

#define HC_SR04_TRIGGER_PIN A4

#define HC_SR04_ECHO_PIN    A5

#define CAL_TRIGGER_PIN 12

#define LED_PIN A6

```

- Se definen los pines utilizados por el sensor ultrasónico y otros componentes como el LED.

 

### Comandos de Movimiento

 

```cpp

#define FORWARD 'F'

#define LEFT 'L'

#define STAND 'S'

#define RIGHT 'R'

#define BACKWARD 'B'

```

- Se definen los comandos que el robot puede recibir para moverse en diferentes direcciones.

 

## 2. Variables Globales

 

```cpp

MiniKame robot;

bool auto_mode = true;  // Inicia en modo autónomo

char cmd = STAND;

```

- Se crea una instancia del robot y se definen variables para controlar el modo (autónomo o manual) y el comando actual.

 

## 3. Configuración Inicial (`setup`)

 

```cpp

void setup() {

  Serial.begin(9600);

#ifdef __DEBUG__

  softSerial.begin(9600);

  softSerial.println("debugging mode");

#endif

  robot.init();

  delay(2000);

  // Calibración del servo

}

```

- Se inicializa la comunicación serie y el robot. Si el modo de depuración está activo, se configura un puerto serie adicional.

 

### Calibración del Servo

 

```cpp

bool state = true;

pinMode(CAL_TRIGGER_PIN, OUTPUT);

digitalWrite(CAL_TRIGGER_PIN, 0);

pinMode(CAL_TRIGGER_PIN, INPUT);

while (digitalRead(CAL_TRIGGER_PIN)) {

  analogWrite(LED_PIN, 128 * state); // LED de calibración

  delay(1000);

  state = !state;

}

```

- Se calibra el servo utilizando un LED para indicar el estado de calibración.

 

## 4. Bucle Principal (`loop`)

 

```cpp

void loop() {

  cur_time = millis();

  if (Serial.available()) {

    char received = Serial.read();

    // Procesar comandos recibidos

  }

  if (auto_mode) {

    // Movimiento automático

  }

  gaits(cmd);

  robot.refresh();

}

```

- El bucle principal verifica si hay comandos disponibles a través del puerto serie y ejecuta acciones basadas en el modo actual (automático o manual).

 

### Procesamiento de Comandos

 

```cpp

if (received == 'X') {

  auto_mode = false;  // Activar modo inalámbrico

  cmd = STAND;

} else if (received == 'x') {

  auto_mode = true;  // Activar modo autónomo

}

```

- Cambia entre modos autónomo e inalámbrico según el comando recibido.

 

## 5. Movimiento Automático

 

```cpp

static char movements[] = {FORWARD, LEFT, OMNI_WALK_R, OMNI_WALK_L, RIGHT, BACKWARD, PUSH_UP, UP_DOWN, HELLO, DANCE, FRONT_BACK, MOON_WALK};

```

- Define un conjunto de movimientos que el robot puede realizar automáticamente.

 

## 6. Funciones de Movimiento (`gaits`)

 

```cpp

boolean gaits(char cmd) {

  switch (cmd) {

    case FORWARD:

      robot.run();

      break;

    case BACKWARD:

      robot.run(0);

      break;

    // Otros casos para diferentes movimientos

  }

}

```

- Esta función recibe un comando y ejecuta el movimiento correspondiente en el robot.

 

## 7. Detección de Obstáculos (si se activa el sensor HC-SR04)

 

```cpp

#ifdef __HC_SR04__

char detect_obstacle(char cmd) {

  long cm = distance_measure();

  if (cm < MIN_DISTANCE) {

    i = STAND;

  } else if (cm >= MIN_DISTANCE && cm <= MAX_DISTANCE) {

    i = BACKWARD;

  }

  return i;

}

```

- Si el sensor ultrasónico está activado, esta función mide la distancia y ajusta el comando del robot si detecta un obstáculo.
