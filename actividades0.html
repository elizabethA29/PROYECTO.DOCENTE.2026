ACTIVIDADES

Ejemplo 1

Encender y apagar un LED con retardo.

Objetivo:

Que el estudiante comprenda cómo usar la función digitalWrite() y delay() para generar una secuencia básica con temporizador.

Código:

void setup() {

  pinMode(13, OUTPUT); // Configuramos el pin 13 como salida

}

void loop() {

  digitalWrite(13, HIGH); // Enciende el LED

  delay(1000);            // Espera 1 segundo (1000 milisegundos)

  digitalWrite(13, LOW);  // Apaga el LED

  delay(1000);            // Espera 1 segundo

}

Ejemplo 2

 Parpadeo del LED usando bucle For.

Objetivo:

Aprender a repetir acciones con una estructura de bucle for, automatizando la secuencia sin repetir código manualmente.

Código:

void setup() {

  pinMode(13, OUTPUT);

}

void loop() {

  for (int i = 0; i < 5; i++) {      // Repite 5 veces

    digitalWrite(13, HIGH);         // Enciende el LED

    delay(500);                     // Espera medio segundo

    digitalWrite(13, LOW);          // Apaga el LED

    delay(500);                     // Espera medio segundo

  }

  delay(2000); // Espera 2 segundos antes de repetir la secuencia

}

Actividad Grupal

“Diseño de secuencia de luces”

Objetivo:

Desarrollar una secuencia de leds  funcional mediante el uso de Arduino IDE y Fritzing.

Descripción de la actividad:

Los estudiantes, se organizan en grupos pequeños de (3 a 4 integrantes), donde diseñan en el programa Fritzing un circuito básico compuesto por tres LEDs conectados a la placa Arduino Nano. Cada LED debe encenderse en una secuencia específica. Una vez diseñado el circuito, los estudiantes programan dicha secuencia utilizando Arduino IDE, y comprobarán si la programación ejecuta correctamente el comportamiento esperado.
